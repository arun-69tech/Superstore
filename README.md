## Project Overview

This project demonstrates the complete analytics lifecycle for a retail sales dataset. It covers data cleaning, exploratory data analysis, SQL analysis, DAX calculations, dashboard development, and business recommendation generation.

The final outcome is a six-page executive Power BI dashboard designed to help stakeholders understand sales performance, profitability patterns, regional trends, customer behavior, and product-level opportunities.

## Problem Statement

Retail businesses generate large volumes of transactional data across customers, regions, product categories, discounts, and order timelines. Without structured analysis, it becomes difficult to identify revenue drivers, detect profitability issues, and make confident business decisions.

This project solves that challenge by converting raw Superstore transactional data into clear, actionable insights for data-driven decision making.

## Project Objectives

- Clean and prepare raw data.
- Perform exploratory data analysis.
- Apply SQL for business questions.
- Develop DAX measures and KPIs.
- Build an interactive Power BI dashboard.
- Generate actionable business recommendations.
- Create a portfolio-ready business intelligence solution.

## Python Analysis Files

The Python workflow is available in both notebook and script format:

- **Superstore_EDA.ipynb** - Original exploratory notebook containing the full step-by-step analysis flow.
- **Phase_1_Data_Understanding.py** - Loads the raw dataset and performs initial data inspection.
- **Phase_2_Data_Cleaning.py** - Cleans the raw data, creates derived date fields, removes duplicates, and exports the cleaned dataset.
- **Phase_3_Python_EDA.py** - Performs visual exploratory data analysis and saves all charts as PNG files inside `Python/Visuals/`.

This structure keeps the notebook useful for presentation and storytelling, while the Python scripts provide a cleaner, reusable workflow for running each phase independently.

## Dashboard Overview

The Power BI report contains six executive dashboard pages:

1. **Executive Summary** - High-level KPIs for sales, profit, orders, customers, and profitability.
2. **Regional Analysis** - Regional and state-level performance comparison.
3. **Customer & Segment Analysis** - Customer contribution, segment performance, and sales distribution.
4. **Product Performance Analysis** - Category, sub-category, and product-level sales and profit analysis.
5. **Sales & Profit Trends** - Time-based trends across months, quarters, and years.
6. **AI Business Insights** - AI-assisted interpretation of key patterns and business recommendations.

## Key Business Insights

- Technology generated the highest sales and profit.
- Consumer segment contributed over 50% of total sales.
- West region emerged as the strongest performer.
- Q4 generated the highest revenue contribution.
- Furniture showed lower profitability despite strong sales.
- Several products and states consistently generated losses.

## Business Recommendations

- Increase investment in high-performing categories.
- Launch loyalty initiatives targeting consumer customers.
- Review pricing and discount strategies.
- Prepare inventory and staffing plans before Q4.
- Investigate loss-making products.
- Address regional profitability issues.
